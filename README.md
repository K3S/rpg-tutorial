# RPG Tutorial

A free, open-source tutorial for modern RPGLE and CL development on IBM i, using VS Code and contemporary tooling.

**Read the tutorial at [rpgtutorial.k3s.com](https://rpgtutorial.k3s.com).**

Sponsored by [King III Solutions](https://k3s.com).

---

## About

A guided journey from zero to writing real RPG programs, designed for developers with no prior IBM i experience.

The tutorial is organized in four parts:

- **Part 1 — Get Set Up.** Free IBM i account at PUB400, install VS Code, connect, create practice tables, run Hello World.
- **Part 2 — Learn the Language.** Free-format RPGLE fundamentals, embedded SQL, procedures, error handling, date math.
- **Part 3 — Build Real Things.** Ten progressive hands-on tutorials using consistent sample data.
- **Part 4 — Going Further.** Curated map to the broader RPG community's resources.

Where excellent community resources already exist — PUB400 signup, connection guides, language references, deeper writing — this tutorial links to them rather than duplicate. Where they don't exist, this tutorial fills the gap. The site's job is to be the map.

## Contributing

This is a community resource and contributions are welcome.

- **Found a typo or a broken link?** Every page on the site has an "Edit this page on GitHub" link at the bottom. Use it to open a quick PR.
- **Want to add a tutorial, cheat sheet, or correction?** Open an issue first to discuss larger changes, or a PR directly for small ones.
- **Something broken?** Open an issue with enough detail to reproduce.

## Tech stack

- [Jekyll](https://jekyllrb.com) with the [Just the Docs](https://just-the-docs.com) theme
- Hosted on GitHub Pages with a custom domain
- Rebuilds automatically on push to `main`
